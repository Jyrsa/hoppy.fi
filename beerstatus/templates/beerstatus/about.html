about:hoppy.fi

Hoppy.fi is an experimental service written by Jyry Suvilehto. Code will eventually be published in GitHub.

This project was sparked my frustration at trying to figure out what beers are available and what are worth buying when I go beer shoppint. Also, I'm fairly certain that I look like an idiot standing there for possibly a long time staring at the counter and making mental judgements.

Possible future feature list

  • Search for the nearest Alko using html5 location API (coordinates exist already)
  • Search for stores containing a particular beer
  • Possibly a comparison view of several potential Alkos
